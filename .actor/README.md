## 📧 Waterfall Email Enrichment

Find verified work email addresses using a multi-step waterfall process across trusted data providers including Apollo, Hunter, Dropcontact, Snov, Prospeo, Skrapp, Clearbit, RocketReach, Datagma, PeopleDataLabs and additional enrichment sources.

Only valid emails are charged. Emails not found or marked invalid are automatically refunded.

---

## 📌 Data Fields Included

| Field | Description |
|---|---|
| first_name | Prospect first name |
| last_name | Prospect last name |
| domain | Company website domain |
| email | Verified work email address |
| email_status | Verification result (valid / invalid / not_found) |

---

## 📊 Sample Output

```json
{
  "first_name": "John",
  "last_name": "Doe",
  "domain": "example.com",
  "email": "john.doe@example.com",
  "email_status": "valid"
}
```

Upload CSV file with the following columns:
